# mobile-app (Expo + TypeScript)

This folder contains the React Native mobile application built with Expo and TypeScript.

## Setup

1. Copy `.env.example` to `.env` and update values.
2. Install dependencies:

```powershell
cd mobile-app
npm install
```

3. Start the app:

```powershell
npm start
```

## Structure

See `src/` for the application code including `components/`, `screens/`, `navigation/`, and `store/`.
